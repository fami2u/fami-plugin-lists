# fami cordova plugin lists
fami官方开发和维护的cordova插件，包括支付宝支付插件，分享插件，消息推送插件等，如果没有找到您需要的，请提交[ISSUE](https://github.com/fami2u/fami-plugin-lists/issues)给我们。
